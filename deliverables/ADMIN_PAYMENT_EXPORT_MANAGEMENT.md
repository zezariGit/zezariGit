# 관리자 결제관리 및 엑셀 다운로드 산출물

## 요구내용
- 관리자페이지에 `결제 관리` 메뉴를 구성한다.
- 결제 관리는 그리드 형태로 `결제번호, 보호자, 대상자, 구분, 결제수단, 결제금액, 결제일`을 조회한다.
- 관리자페이지의 각 그리드에는 엑셀 다운로드 기능을 제공한다.

## 반영내용
- 기존 `결제 관리` 메뉴를 운영형 화면으로 개편했다.
- 상단 현황 카드는 `총 매출액`, `주문 매출`, `구독 매출`, `광고 매출`, `취소/환불 비용`을 보여준다.
- 결제내역은 최대 300건을 최신순으로 조회한다.
- 결제내역 통합 검색은 결제번호, 보호자명, 보호자 연락처, 보호자 이메일, 대상자명, 결제 상품, 결제수단을 대상으로 한다.
- 조회 필터는 `거래구분`, `결제상태`, `기간`으로 구성했다.
- 결제 그리드는 `선택`, `주문/구독/광고 번호`, `거래구분`, `대상자(보호자)`, `거래일시`, `금액`, `결제상태`, `관리` 컬럼으로 구성했다.
- 오른쪽 상세정보 카드는 `결제 정보`와 `대상자정보`를 구분해 표시한다.
- 상세정보 카드의 `취소/환불` 버튼을 누르면 팝업에서 거래구분, 환불금액, 환불사유를 입력할 수 있다.
- 취소/환불 이력은 `payment_refunds` 테이블에 저장한다.
- 환불금액은 이미 접수된 금액까지 포함해 원 결제금액을 넘지 못하도록 검증한다.
- `이용권 옵션 가격` 설정 화면은 결제내역 아래에 유지했다.
- 관리자 그리드의 다운로드는 브라우저에서 UTF-8 BOM 포함 CSV를 생성한다.
- CSV 파일은 Excel에서 바로 열 수 있으며, 필터 적용 후 화면에 조회된 결과 기준으로 내려받는다.

## 결제내역 데이터 기준
- 상품/이용권 결제는 `product_orders`를 기준으로 조회한다.
- 상품 단독 구매는 `주문`, 이용권 구매는 `구독`으로 표시한다.
- 결제상태는 `결제완료`, `결제대기`, `결제취소`, `환불접수`, `결제실패`로 표시한다.
- 광고는 현재 `subject_ads`의 광고 금액 이력을 함께 표시한다.
- 광고 결제수단은 실제 Meta/Toss 광고 결제 API 연동 전 단계이므로 `광고 결제 준비`로 표시한다.
- 취소/환불 비용은 `payment_refunds`의 접수/완료 금액을 합산한다.

## 엑셀 다운로드 적용 범위
- 결제 관리: 결제내역
- 보호자 관리: 보호자 목록
- 관리대상자 관리: 관리대상자 목록
- QR 관리: 생성된 QR 목록
- 관리자 관리: 가입된 사용자 목록
- 상품 관리: 상품 목록
- 주문/배송 관리: 주문 목록
- 광고 관리: 광고 목록
- 실종신고 관리: 실종신고 목록
- 위치공유 관리: 위치공유 목록
- 고객문의: 고객문의 목록

## 운영 메모
- 현재 다운로드 파일은 `.csv` 확장자이지만 Excel에서 바로 열 수 있는 UTF-8 CSV이다.
- `.xlsx`가 반드시 필요해지면 서버 또는 클라이언트에 별도 스프레드시트 라이브러리를 추가하는 방식으로 확장 가능하다.
- 광고 결제내역은 광고 API와 실제 광고 결제 흐름을 붙인 뒤 결제수단/결제상태 컬럼을 실제 결제 데이터로 대체하는 것이 좋다.
- 현재 `취소/환불`은 관리자 운영 기록과 내부 상태 변경을 수행한다.
- 실제 토스 결제 취소/환불 API 호출은 운영 승인, 환불 정책, 권한 확인 후 별도 연결하는 것이 안전하다.

## 이미지 생성 프롬프트
> REAL_QR_FIND 관리자 콘솔 화면. 왼쪽 세로 메뉴와 숨김 토글이 있고, 결제 관리 화면이 보인다. 상단에는 총 매출액, 주문 매출, 구독 매출, 광고 매출, 취소/환불 비용 카드가 있다. 그 아래 검색어, 거래구분, 결제상태, 기간 필터와 초기화/검색 버튼이 있다. 본문은 왼쪽 결제 그리드와 오른쪽 상세정보 카드로 나뉘며, 그리드에는 주문/구독/광고 번호, 거래구분, 대상자(보호자), 거래일시, 금액, 결제상태, 관리 컬럼이 있다. 오른쪽 카드에는 결제 정보와 대상자정보가 표시되고, 하단 취소/환불 버튼을 누르면 환불금액과 환불사유 입력 팝업이 열린다. 공공서비스 스타일의 흰색 배경, 얇은 회색 테이블 선, 보라색 포인트 버튼, 업무용 SaaS 대시보드 느낌.
