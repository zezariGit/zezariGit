# 관리자 운영형 대시보드 개편 산출물

## 요구내용
- 관리자 대시보드를 첨부 시안처럼 현황 중심 화면으로 변경한다.
- 스타일뿐 아니라 기능적으로도 운영자가 바로 확인해야 할 지표, 그래프, 최근 현황, 주문/광고/구독/매출 요약을 한 화면에 구성한다.

## 반영내용
- 상단 `현황` 영역을 5개 카드로 재구성했다.
  - 전체 보호자 / 전체 대상자
  - 신규 보호자 / 신규 대상자 및 전일 대비
  - 활성 QR / 미활성 QR 및 대상자 대비 비율
  - 광고 진행중 / 광고 심사반려
  - 일 매출 / 월 매출 및 전일·전월 대비
- 최근 30일 추이 그래프를 추가했다.
  - 일별 매출 현황
  - 일별 주문 현황
  - 일별 위치공유·광고 활동 현황
- `최근 현황`을 표 형태로 개편했다.
  - 신규 가입자
  - 신규 주문
  - 최근 활동 알림
  - 주의 대상 현황
- 하단 운영 현황을 추가했다.
  - 주문 현황: 신규주문, 배송준비, 배송중, 배송완료
  - 최근 실종 광고
  - 광고 진행 현황 도넛 그래프
  - 구독 현황 도넛 그래프
  - 매출 현황 표

## 데이터 기준
- 보호자: `guardians`
- 관리대상자: `subjects`
- QR: `qr_codes`
- 주문/상품/구독 매출: `product_orders`
- 광고 매출 및 광고 상태: `subject_ads`
- 구독 상태: `subscriptions`
- 최근 활동 알림: `guardian_notifications`
- 위치공유 활동: `location_shares`
- 고객문의: `customer_inquiries`

## 주요 집계 기준
| 영역 | 기준 |
| --- | --- |
| 신규 보호자/대상자 | 한국시간 오늘 생성 건수와 어제 생성 건수 비교 |
| 활성 QR | `qr_codes.is_active = 1` 이고 `activated_at`이 있는 QR |
| 미활성 QR | `activated_at`이 없거나 관리자 비활성인 QR |
| 일 매출 | 오늘 결제 완료 주문 매출 + 오늘 광고 생성 금액 |
| 월 매출 | 선택 월 상품 매출 + 구독 매출 + 광고 매출 |
| 환불 금액 | 선택 월 취소/실패 주문 금액 |
| 주의 대상 | QR 미활성 10일 초과, 배송준비 2일 이상, 구독만료 7일 이내, 광고 심사 지연, 취소/환불 대기 |

## 구현 파일
- `lib/db.js`
  - `getAdminDashboardData()` 확장
  - 전일 대비, 월 매출, 최근 30일 추이, 주의 대상, 주문/광고/구독 상태 조회 추가
- `app/admin/page.js`
  - 운영형 대시보드 JSX 구성
  - SVG 선 그래프, 표 컴포넌트, 주문 흐름, 도넛 패널 추가
- `app/globals.css`
  - 5개 현황 카드, 그래프, 표, 도넛 패널, 하단 운영 현황 스타일 추가

## 검증
- `npm run build` 성공.
- 직접 Node import 검증은 Next.js의 확장자 없는 ESM import 해석 차이로 수행하지 못했다.
- Next.js 빌드 기준으로 앱 번들 문법과 라우트 생성은 정상 확인했다.

## 운영 참고사항
- 방문자 분석 전용 테이블이 아직 없으므로 그래프의 세 번째 활동 지표는 위치공유 및 광고 생성 활동으로 구성했다.
- 실제 방문자 수 집계가 필요하면 추후 `page_views` 또는 Vercel Analytics 연동 데이터를 별도 테이블/조회로 추가하면 된다.
- 광고 심사반려는 `subject_ads.meta_status`에 `reject`가 포함된 데이터를 기준으로 계산한다.

## 이미지 생성 프롬프트
> REAL_QR_FIND 관리자 운영 대시보드 화면. 상단에 "현황" 제목과 5개의 두꺼운 검정 테두리 카드가 있다. 각 카드에는 큰 검정 아이콘과 한국어 지표가 표시된다: 전체 보호자/전체 대상자, 신규 보호자/신규 대상자, 활성 QR/미활성 QR, 광고 진행중/광고 심사반려, 일 매출/월 매출. 중앙에는 최근 30일 매출·주문·위치공유 활동 선 그래프와 노란 설명 메모가 있다. 그 아래에는 신규 가입자, 신규 주문, 최근 활동 알림, 주의 대상 현황 표가 배치된다. 하단에는 주문 현황 흐름, 최근 실종 광고 표, 광고 진행 도넛 그래프, 구독 현황 도넛 그래프, 매출 현황 표가 있다. 업무용 SaaS 관리자 화면, 흰색 배경, 검정 테두리, 절제된 보라색/파란색/노란색 강조, 한국어 텍스트, 16:9 발표 슬라이드.
