# 관리자 세로 메뉴 구현 산출물

## 구현 목적
- 관리자 페이지 상단의 가로 탭을 왼쪽 세로 메뉴로 변경한다.
- 관리자가 필요할 때 메뉴를 숨겨 넓은 작업 영역을 사용할 수 있게 한다.
- 모바일에서는 화면 폭을 침범하지 않도록 메뉴와 본문을 한 열로 배치한다.

## 구현 내용
- `app/admin/admin-workspace.js`
  - 관리자 메뉴와 콘텐츠 레이아웃을 담당하는 클라이언트 컴포넌트
  - 메뉴 숨기기/펼치기 버튼 제공
  - `localStorage`에 접힘 상태 저장
  - 현재 선택한 관리 메뉴 강조
- `app/admin/page.js`
  - 기존 관리 기능을 `AdminWorkspace` 안에 배치
  - 각 메뉴 URL과 서버 데이터 조회 방식은 기존과 동일하게 유지
- `app/globals.css`
  - 데스크톱: 216px 세로 메뉴와 본문 2열 구성
  - 접힘 상태: 메뉴 영역을 56px로 축소하고 메뉴 항목 숨김
  - 860px 이하: 메뉴와 본문을 한 열로 배치
  - 키보드 포커스, 호버, 모션 감소 설정 반영

## 메뉴 구성
1. 보호자 관리
2. 주문/배송
3. QR 관리
4. 관리자 관리
5. 결제 관리
6. 상품 관리
7. 광고 관리

## 검증 결과
- Next.js 프로덕션 빌드 성공
- 데스크톱 펼침 메뉴 폭: 216px
- 데스크톱 접힘 메뉴 폭: 56px
- 새로고침 후 접힘 상태 유지 확인
- 활성 메뉴 `주문/배송` 강조 확인
- 모바일에서 메뉴 아래에 본문 배치 확인
- 데스크톱과 모바일 모두 가로 스크롤 없음
- 운영 DB와 분리된 임시 SQLite DB에서 관리자 로그인 및 화면 검증
- 테스트 DB와 화면 캡처 파일 삭제 완료

## 발표자료용 이미지 생성 프롬프트
> 한국 공공서비스 스타일의 웹 관리자 대시보드 비교 이미지. 왼쪽 장면은 펼쳐진 216px 세로 메뉴에 보호자 관리, 주문/배송, QR 관리, 관리자 관리, 결제 관리, 상품 관리, 광고 관리 항목이 정돈되어 있고 주문/배송이 파란색으로 활성화되어 있다. 오른쪽 장면은 메뉴가 56px 접힘 버튼만 남기고 본문 작업 영역이 넓어진 상태다. 흰색 카드, 옅은 회색 배경, 절제된 파란색 강조색, 실제 운영 도구처럼 차분하고 전문적인 UI, 한국어 텍스트, 16:9 발표 슬라이드용.
